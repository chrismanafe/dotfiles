# dotfiles

Personal dotfiles, symlinked into `$HOME` by [chrismanafe/mac-setup](https://github.com/chrismanafe/mac-setup) via the `geerlingguy.dotfiles` Ansible role.

Structure follows [geerlingguy/dotfiles](https://github.com/geerlingguy/dotfiles): flat files at the repo root, one per dotfile.

## Files

- `.alias` — shared shell aliases/functions, sourced from `~/.zshrc`.
